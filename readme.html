[Rol]

Actúa como un desarrollador front-end experto y diseñador de interfaces interactivas especializado en visualización de datos académicos y educativos.

[Instrucción]

Crea la estructura completa (HTML con CSS embebido en el mismo archivo) para una infografía interactiva web de tipo "Estadística / datos". El objetivo es mantener un diseño visualmente muy limpio, minimalista y de aspecto premium (estética Glassmorphism) en su estado inicial, mostrando los íconos SVG y los títulos. Oculta las cifras y los gráficos estadísticos, y revélalos únicamente cuando el usuario interactúa con cada tarjeta (hover).

[Contenido y Personalización]
  • Diseño Visual: Paleta HEX Principal #1A4A5E, Secundario #2A9D8F, Acento #D35400.
  • Datos Estadísticos: 5 Tarjetas (Dona para dieta, Barras para eficiencia, Boxplot para cortisol, Línea para crecimiento y Dispersión para nitrógeno).
  • Pie de Página: Fuente bibliográfica de Silva-Villaizan, A. (2026) y firma "Diseñado por: Alexandra Yanira Silva Villaizan".
[Pasos]
  1. Estructura Base (HTML): Contenedor Grid de 3 columnas.
  2. Activos: Integración de íconos SVG personalizados (Peces, alimentación, red, etc.).
  3. Efectos: Implementación de hover con desenfoque y transiciones suaves.
  4. Gestión: Todo el código CSS dentro de etiquetas de estilo.
[Fin]

Devuelve ÚNICAMENTE el código fuente. El resultado final debe ser un solo bloque de código en un documento index.html funcional.

[Restricciones]
  • Utilizar únicamente CSS puro (Vanilla CSS).
  • Sin imágenes externas; todo mediante código SVG nativo.
  • Diseño estrictamente responsivo (Mobile First).
  • Idioma: Español técnico.
